The Users API endpoints include:

GET /api/v1/users - Get all users (admin only)
GET /api/v1/users/:id - Get single user (admin only)
POST /api/v1/users - Create new user (admin only)
PUT /api/v1/users/:id - Update user (admin only)
DELETE /api/v1/users/:id - Delete user (admin only)
PUT /api/v1/users/:id/photo - Upload user photo
GET /api/v1/users/stats - Get user statistics (admin only)
PUT /api/v1/users/:id/deactivate - Deactivate user (admin only)
PUT /api/v1/users/:id/reactivate - Reactivate user (admin only)
GET /api/v1/users/:id/activity - Get user activity log (admin only)
